# NNIT Real Estate backend

This Express backend exposes API endpoints for the NNIT Real Estate frontend.

## Run locally

1. Open a terminal in `C:\NNIT Real Estate app`
2. Run `npm run backend`
3. The backend will listen on `http://localhost:4000`

## API endpoints

- `GET /api/properties` — returns a list of sample property listings
- `POST /api/contact` — accepts JSON contact requests with `name`, `email`, and `message`
